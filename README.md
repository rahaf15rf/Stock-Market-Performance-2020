## Overview
This project showcases a professional stock market performance analysis for the year 2020. The dashboard provides a visual breakdown of market trends, trading volumes, and price fluctuations for several major companies.

**Note:** The data used in this project is **synthetic (dummy data)** and was generated for the purpose of demonstrating data analysis, dashboard design, and visualization skills. It does not represent actual market performance or real-world financial data.

## Dashboard Features
- **Number of Records:** A visual summary of data density per company.
- **Trading Volume by Company:** A bar chart comparing total trading activity across different entities.
- **Stock Performance:** A line chart tracking the average closing price trends throughout the year 2020.
- **High vs. Close Price Analysis:** An area chart visualizing the relationship between daily high and closing prices.
- **Monthly Price Range Analysis:** A scatter plot showing the monthly distribution of high and low price points.
- **Opening Price Distribution:** A bar chart highlighting the average opening prices for each company.

## Tools & Technologies
- **Data Analysis & Visualization:** Tableau
- **Data Context:** Financial Market Analysis (Simulation)
